```css
::-webkit-scrollbar {
  width: 0.625rem;
}

::-webkit-scrollbar-thumb {
  background: rgba(0, 0, 0, 100);
}

::-webkit-scrollbar-track {
  background: rgba(241, 241, 241, 100);
}
```
